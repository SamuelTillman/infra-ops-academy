# Contributing to The Open Source Infrastructure & Operations Academy

First off, thanks for taking the time to contribute! This project thrives on community input.

## How Can I Contribute?

### Suggest a Certification or Resource
- Open an [issue](https://github.com/SamuelTillman/infra-ops-academy/issues) with the certification name, provider, cost, and which skill track it belongs to
- Include a link to the official certification page

### Submit a Pull Request

1. Fork the repo
2. Create a branch (`git checkout -b add-new-cert`)
3. Make your changes to `README.md`
4. Commit your changes (`git commit -m "Add XYZ certification to Cloud Platforms"`)
5. Push to your branch (`git push origin add-new-cert`)
6. Open a Pull Request

### What We're Looking For

- New certifications with accurate pricing and official links
- New skill tracks (Azure, GCP, etc.)
- Book recommendations with author names
- Community resources (subreddits, Discord servers, podcasts)
- Corrections to pricing, links, or certification names
- Money saving tips for exam prep

### Guidelines

- Resources must be free or clearly priced
- Links must point to official sources (no pirated content)
- Certifications must be currently available (not retired)
- Keep descriptions concise
- Follow the existing formatting style in the README

### What NOT to Submit

- Paid courses or bootcamps disguised as free resources
- Affiliate links
- Self-promotional content
- Duplicate entries
