# Free or Low Cost Resources

!!! info
    Free or low cost courses, YouTube channels, and hands-on labs to prep for certifications without spending a dime.

## Free Courses & Platforms

- [freeCodeCamp](https://www.freecodecamp.org/) — Full courses on Linux, Python, cloud, networking, and more
- [freeCodeCamp YouTube](https://www.youtube.com/@freecodecamp) — Full-length certification prep courses (AWS, Docker, Kubernetes, Terraform, Python, and more)
- [AWS Skill Builder](https://skillbuilder.aws/) — Free AWS training courses and exam prep (some paid content)
- [Microsoft Learn](https://learn.microsoft.com/) — Free GitHub certification prep and cloud training
- [KodeKloud Free Tier](https://kodekloud.com/) — Hands-on DevOps labs (free tier available)
- [Anthropic Academy](https://anthropic.skilljar.com/) — Free AI/ML and Claude courses with completion certificates
- [Google SRE Books](https://sre.google/books/) — All three SRE books free to read online
- [Linux Journey](https://linuxjourney.com/) — Free interactive Linux learning from beginner to advanced
- [The Odin Project](https://www.theodinproject.com/) — Free full-stack curriculum with Git and command line foundations
- [edX](https://www.edx.org/) — Free courses from MIT, Harvard, and Linux Foundation (paid certificates optional)
- [Coursera](https://www.coursera.org/) — Free to audit courses from Google, IBM, and top universities (paid certificates optional)
- [NVIDIA Deep Learning Institute](https://www.nvidia.com/en-us/training/) — Free self-paced AI/ML courses
- [Cisco Networking Academy](https://www.netacad.com/) — Free networking and cybersecurity courses
- [Professor Messer](https://www.professormesser.com/) — Free CompTIA A+, Network+, and Security+ video courses
- [AWS Workshops](https://workshops.aws/) — Free guided AWS hands-on workshops
- [Red Hat Developer](https://developers.redhat.com/) — Free RHEL developer subscription and learning resources
- [Python.org Tutorial](https://docs.python.org/3/tutorial/) — Official Python tutorial, free forever
- [Postman Academy](https://academy.postman.com/) — Free API fundamentals, testing, and automation courses with certificates

## Practice Exams & Labs

- [Killer.sh](https://killer.sh/) — CKA, CKAD, CKS practice exams (included with exam purchase from Linux Foundation)
- [AWS Well-Architected Labs](https://wellarchitectedlabs.com/) — Free hands-on AWS labs
- [Play with Docker](https://labs.play-with-docker.com/) — Free Docker sandbox in the browser
- [Play with Kubernetes](https://labs.play-with-k8s.com/) — Free Kubernetes sandbox in the browser
- [HashiCorp Learn](https://developer.hashicorp.com/terraform/tutorials) — Free Terraform tutorials and labs
- [Katacoda Alternatives on Killercoda](https://killercoda.com/) — Free interactive DevOps scenarios (Linux, K8s, Docker, Git)
- [OverTheWire](https://overthewire.org/wargames/) — Free Linux and security challenges (great for beginners)
- [sadservers.com](https://sadservers.com/) — Free Linux troubleshooting scenarios (like LeetCode for sysadmins)
- [Datadog Learning Center](https://learn.datadoghq.com/) — Free Datadog courses and certification prep
- [New Relic University](https://learn.newrelic.com/) — Free observability training and certification prep
